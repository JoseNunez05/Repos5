Assertions
----------

1) implicit <-- built in and supported by cypress directly

    .should()
        eq
        contain
        include
        exist
        have.length
        have.value
        be.checked
        not.be.checked
        have.text

    .and()
        eq
        contain
        include
        exist
        have.length
        have.value
        be.checked
        not.be.checked
        have.text

2) explicit <-- need to write explicity

    .expect(value)
        .to.equal
        .to.notEqual

    .assert(value, otherValue)
        .equal
        .notEqual
